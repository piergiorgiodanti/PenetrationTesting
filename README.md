# Report Penetration Testing

Nel file `report.pdf` è presente l'analisi VAPT effettuata sul docker `gabrielec/ptexam1`.

I programmi riportati permettono di eseguire gli exploit descritti nell'elaborato in maniera automatizzata e riproducibile.

Per eseguire tali programmi utilizzare `uv run python <nome_programma.py>`.
